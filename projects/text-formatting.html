Text Formatting

In this page basic html formatting tags will be used to edit text within the html doc.

Visually some of these html elements do the same thing. However tags such as em of strong have add meaning or importance when it comes to showing emphases or strong importance or in a body of text.


I can write em or strong and have them look the same as above. Though the tags that where used were not em or strong but i and b

i = italic b = bold; These tags carry different meaning and are not interpreted the same way when using a "screen reader" for example the i tag adds a alternative voice or mood to a text.
